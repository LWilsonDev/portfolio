About

I am a full stack web and mobile developer specialising in React Native.

I am currently working as a Frontend Developer at Estendio, where I have a key role in the creation and maintenance of our mobile application Present Pal as well as our accompanying software for Chrome and Powerpoint built with React. Prior to this I was a software engineer at Sales Agility where I worked on client projects based on the popular open-source SuiteCRM


Alongside my 'proper job' I like to build small side projects to explore different aspects of web development. Most recently this has included building mini apps for React Native, focusing on some of the key areas of animation and gesture handling. A few of my experiments for mobile and web are showcased below...

NASA App

A React Native mobile app to show NASA's Astronomy Picture of the Day. Users can search for images by date, double-tap to zoom, single tap to view menu and reveal an animated information tab. Users can add their favourite images to view again later.

Kitchen Helper App

This is a single-page mobile app exploring animation using the Lottie Library. Users can enter an amount in weight and select which unit to convert it to. When the conversion starts, a playful animation runs. I built the animation using Figma to tweak an existing SVG, and Haiku Animator to prepare the Lottie animation. (Play video above to view the animation)

Beethoven Music Memory

Having previously built a music memory game to help children learn to differentiate between musical pitches, I was approached by a music teacher in Germany to extend and adapt the game for his students. The goal was to celebrate the 250th year since Beethoven's birth by helping the students learn some of the composer's most famous works.

The Beer List

This is a Django web app where users can log in securely, keep track of their favourite beers, and browse other user's profiles for inspiration. With the growing trend for craft beer and independent breweries, I wanted to create an app where users can review the beers they have tried, and keep track of their favourites. I'm intending to overhaul this site and turn it into a fully fledged React Native mobile app at some point!

Space API's

This was the start of my exploration into NASA's API's. This is a Django project built using Python, utilising AJAX to fetch real-time data about the International Space Station. You can find the current position of the ISS above the Earth, and see the location on Google Maps, as well as find out how many people are currently in space
